# Numpy Image Manipulation
This project involves image manipulation using Numpy in Python.

## Directory Structure
- `data/`: Contains input images for the project.
  - `input/`: Place your images here.
- `notebooks/`: Jupyter notebooks for code exploration.
- `scripts/`: Python scripts for image manipulation.
- `README.md`: Project description and setup instructions.
- `environment.yml`: Dependencies for the project.
